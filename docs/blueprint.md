# **App Name**: Emperor's Choice

## Core Features:

- Dynamic Menu & Cart System: Customer-facing interface to browse categorized menu items, view detailed descriptions, customize orders (e.g., spice level, special instructions), add items to a persistent shopping cart, and manage quantities.
- Secure User Accounts & Checkout: User authentication and registration flow, including name, phone number, and password. Allows users to review their cart, confirm pickup time, place orders, and view past orders in their history section. Payments are handled at the restaurant.
- Admin Menu & Content Management: A secure admin interface for adding, editing, and deleting menu items, including names, descriptions, prices, images, and managing menu categories. Also allows updates to restaurant information.
- Admin Order Management Dashboard: Dashboard for administrators to view incoming orders in real-time, see full order details, update order statuses (e.g., 'Received', 'Preparing', 'Ready for Pickup'), and manage customer records.
- Automated SMS Notification System: Integration with an SMS gateway (e.g., Twilio) to automatically send text message notifications to customers when their order status is updated to 'Ready for Pickup', including estimated pickup time.
- Info Page & External Links: A dedicated page providing essential restaurant information like address, hours of operation, contact number, and clear instructions on how to order. Includes prominent, clickable links to DoorDash (https://www.doordash.com/store/emperor's-choice-chinese-restaurant-clayton-555783/721362/) and UberEats (https://www.ubereats.com/store/emperors-choice-chinese/LSvUUpr3XReBK4G-QPLLuw) for delivery services.
- AI Item Description Assistant: An AI-powered tool within the admin panel that can suggest creative and enticing descriptions for new or existing menu items, based on basic input such as dish name and ingredients.

## Style Guidelines:

- Primary color: A warm, rich reddish-brown (#C33D22) to evoke the restaurant's name and create a welcoming, traditional, yet modern feel, ensuring strong contrast on lighter backgrounds.
- Background color: A subtle, light off-white with a warm undertone (#F5F4F0) derived from the primary hue, providing a clean, bright, and highly readable canvas for menu items and text.
- Accent color: A vibrant gold-orange (#F4AF25) to highlight key actions, interactive elements, and add a touch of regality and visual energy to the design, creating effective contrast with both primary and background colors.
- Headlines and prominent titles: 'PT Sans' (sans-serif) for its blend of modernity and humanist warmth, ensuring headings are inviting and legible.
- Body text and descriptions: 'Inter' (sans-serif) for its high readability and neutral, objective appearance, ideal for menu details and information-heavy sections.
- Use simple, recognizable line icons for navigation, shopping cart, and account management, maintaining a clean and modern aesthetic. Utilize high-quality food photography prominently for menu items to enhance appeal.
- Implement a fully responsive design that adapts seamlessly to various screen sizes (mobile, tablet, desktop) with intuitive navigation, clearly organized menu categories, and a streamlined checkout flow for optimal user experience across all devices.
